# About

Contains different explorations in areas of maths I've studied or come across.

- `approximating-pi.ipynb`: approximating $\pi$ using inscribed and circumscribed polygons.
- `euclidean-algorithm.ipynb`: the Euclidean algorithm, its complexity, and Bezout's lemma.
- `euler-product-formula.ipynb`: the Euler product formula and the Basel problem.

- `proof-assistant-essay`: my essay for the Caius Explore on the proof assistant Coq.
